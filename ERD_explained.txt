
Games:

this table reprents all games available in the gameshop, this data has the name, genre and price of the game relaterd with the publisher.

game_inventory:
this is the data that keeps track of the inventory of all the games available


game_company:
game company is the publisher of the game. it has the company name, location.


orderitems:

this has the order details of each selected item, it has the game id,with price and quantity

orders:

an order can have many order items, then puts all of them together as an order once confirmed with an order date and to which customer it is following customer_id.


customers:

this has the customers information needed to ship the orders, it has cuntomer name lastname and address/phone numbere

shipments:

this is all the information needed to ship the order, it has the order_id with  destination and will be shipped following th shipment_date
